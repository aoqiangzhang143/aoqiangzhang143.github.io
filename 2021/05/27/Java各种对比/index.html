==和equals对比

  1. ==是判断两个变量是否指向同一块内存空间；equals是判断两个变量或实例所指向的内存空间的值是否相同
  2. ==是对内存地址进行比较，equals则是对内容进行比较；
  3. ==比较引用是否相同，equals比较值是否相同
